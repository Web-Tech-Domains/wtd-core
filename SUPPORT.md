# Support

WTD Core is an open-source framework project. Please choose the right support path so maintainers can respond efficiently.

## Questions

Use GitHub Discussions when available, or open a question issue with:

- What you are trying to build.
- Your PHP version.
- Relevant config, route, model, or controller code.
- Exact command output or error text.

## Bugs

Open a bug report with:

- Steps to reproduce.
- Expected behavior.
- Actual behavior.
- Minimal code sample.
- Output from `php core diagnostics` when relevant.

## Security

Do not report vulnerabilities publicly. Use [SECURITY.md](SECURITY.md).
