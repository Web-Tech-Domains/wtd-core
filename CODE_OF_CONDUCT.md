# Code of Conduct

WTD Core uses the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

We pledge to make participation in this project a respectful, harassment-free experience for everyone, regardless of background, identity, experience level, or viewpoint.

## Expected Behavior

- Be respectful and constructive.
- Focus on the technical problem and project goals.
- Accept feedback professionally.
- Avoid personal attacks, harassment, or discriminatory language.

## Unacceptable Behavior

- Harassment, threats, or intimidation.
- Insults, trolling, or sustained disruption.
- Publishing private information without permission.
- Any behavior that would reasonably make contributors feel unsafe.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior violates this code.

Report conduct concerns privately to:

```text
info@webtechdomains.in
```
